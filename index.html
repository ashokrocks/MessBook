MessBook

Its a project based on SWT (Standard Window Toolkit) JAVA.

Its a Hostel mess management Desktop software modeled on the real world working of a real mess payment system. The application is multi threaded and can run on

1.Windows (all versions).

2.Linux (tested on Ubuntu & elementary os(GTK 2)).

It also prints the recipt.

Import Messbook.sql in your MySql DB and run the respective file. (Schema name- mydb)
